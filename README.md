# aerodynamics-iterative-method
Итерационный метод расчёта тяги вращающейся лопасти в режиме зависания. Реализация на C++. Позволяет снизить погрешность с 60–100% до 5–10% относительно ANSYS CFX.
##
Данный репозиторий содержит реализацию полуаналитического итерационного метода расчёта аэродинамических характеристик вращающейся лопасти летательного аппарата в режиме зависания. Разработан в рамках выпускной квалификационной работы (ЮФУ, 2026).

## Содержание
- Исходный код на C++
- Расчёт силы тяги с учётом индуцированной скорости
- Реализация итерационного алгоритма с последовательным уточнением эффективного угла атаки
- Сравнение с ANSYS CFX

## Основные результаты
- Погрешность снижена с 60–100 % до 5–10 %
- Сходимость достигается за 6–25 итераций
- Подтверждено на примере DJI Mavic Air

## Стек
- C++ (Visual Studio)
- Численное интегрирование по радиусу лопасти
